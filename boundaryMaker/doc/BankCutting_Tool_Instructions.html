Overview

The BC Tool is an AutoLISP utility for Civil 3D that inserts pre-defined bank cutting blocks with the correct Height (H) and Distance (D) values — picked directly from the drawing using a dialog interface.

It calculates height from two text labels, measures perpendicular distance to a polyline, and inserts the matching block variant in one click.

02 —

Loading the Tool

1
Copy files to AutoCAD Support folder
Place both bank_cutting.lsp and bank_cutting.dcl in the same AutoCAD Support path folder
2
Load the LSP file
AutoCAD → APPLOAD → select bank_cutting.lsp → Load
3
Register the blocks (first time only)
Insert all four block variants from Tool Palette at least once so they exist in the drawing definition
4
Run the command
Type BC in the Command Bar and press Enter — the dialog opens
03 —

Step-by-Step Workflow

1
Pick Height — click "PICK HIGHT"
Select First TEXT/MTEXT (BE or FT level) → then Second TEXT/MTEXT (TOE or GL level). Difference is auto-calculated as the H value.
2
Pick Distance — click "PICK DISTRANCE"
Click the center point in the drawing → then select the nearby track Polyline or Line. Perpendicular distance is measured and set as D value.
3
Set Direction — click UP or DN
Choose UP if the bank cutting is above the track, DN if below. This controls which block variant is inserted.
4
Insert — click "PICK ME"
Click anywhere in the drawing — the block inserts at that point with H and D values written in. Values then clear automatically for the next block.
H-LG case: If the second text value is higher than the first (negative height), H is set to "H-LG" automatically and the LG block variant is used. The D label is hidden in this case.
04 —

Dialog Buttons

PICK HIGHT

Pick two TEXT or MTEXT objects. Calculates height difference (First − Second) and sets the H value. Enables distance and direction buttons.

Always first step
PICK DISTRANCE

Click a center point, then select a Polyline or Line. Measures the closest perpendicular distance and sets the D value.

Enabled after Height is set
UP

Sets direction to UP. The tool will insert the bank_cutting_up or bank_cutting_up_lg block variant.

Enabled after Height + Distance
DN

Sets direction to DN. The tool will insert the bank_cutting_dn or bank_cutting_dn_lg block variant.

Enabled after Height + Distance
PICK ME

Click any point in the drawing — block inserts there with H and D filled automatically. Values clear for the next block.

Enabled after direction is set
CLOSE

Closes the Bank Cutting dialog and exits the BC command.

Available anytime

Block Variants

Block NameDirectionH ValueD Label
bank_cutting_up UP Normal (e.g. H-2.50) Shown (e.g. D-4.80)
bank_cutting_dn DN Normal (e.g. H-2.50) Shown (e.g. D-4.80)
bank_cutting_up_lg UP H-LG Hidden (deleted)
bank_cutting_dn_lg DN H-LG Hidden (deleted)
Important: All four blocks must exist in the drawing before running BC. Insert each from the Tool Palette once to register them. The tool alerts you if a required block is missing.
