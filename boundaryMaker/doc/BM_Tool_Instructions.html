User Guide  ·  Command: SB

01 —

Overview

The BM Tool is an AutoLISP utility for Civil 3D that automatically generates boundary lines along a selected Polyline using Chainage and Left/Right offset values read from a CSV file.

Left boundary drawn in Red
Right boundary drawn in Green
Layer "1-BOUNDARY" auto-created
JOINCOLORLINES runs automatically
02 —

CSV File Format

The CSV must have a header row as the first line. Each data row requires three columns:

ChainageLeft (L)Right (R)
0.0005.5004.200
10.0006.0004.500
20.0005.8005.000
30.0006.2004.800
Note: Each Chainage value must be greater than the Starting Chainage entered. Rows with a lower value are skipped automatically.
03 —

How to Use

Type BM in the AutoCAD Command Bar and press Enter.


1
Load the LSP File
AutoCAD → APPLOAD → select CivilBoundaryTool_v6.lsp
2
Run the Command
Type BM in the Command Bar and press Enter
3
Select the Polyline
Click on the alignment polyline in your drawing
4
Enter Starting Chainage
Type the starting chainage value and press Enter (usually 0)
5
Select CSV File
Browse and select your CSV file from the file dialog
6
View the Output
Left (Red) and Right (Green) boundary lines are drawn automatically
04 —

Commands

CommandDescription
BM Runs the full boundary process — prompts for Polyline, Starting Chainage, and CSV file
JOINCOLORLINES Joins existing Red and Green lines in the drawing into separate polylines (can also be run standalone)
05 —

Output

Left Boundary
Red polyline · Color 1 · Layer 1-BOUNDARY
Right Boundary
Green polyline · Color 3 · Layer 1-BOUNDARY
Offset Lines
Perpendicular lines at each Chainage point
Labels
L=xx.xxx and R=xx.xxx text at midpoints

Troubleshooting

Error MessageSolution
Selected object polyline illai! Select only an LWPOLYLINE or POLYLINE object
CSV data illai or read error Check the CSV is valid and has a header row as the first line
Chainage XX – skip (dist < 0) Verify the Starting Chainage is less than all CSV chainage values
Left/Right points 2-ku kuraivu Ensure the CSV has at least 2 valid data rows
Red/Green lines: Not enough endpoints Confirm BM has already run and lines exist in the drawing
Compatibility: Civil 3D 2020 and later. AutoLISP support must be enabled in AutoCAD settings.
